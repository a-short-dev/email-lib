## [Unreleased]

### Added
- Initial release of Weaver email library
- React components (Html, Button, Text)
- Vue.js components with same API
- Tailwind CSS support with automatic inline style conversion
- Visual email builder with import/export
- Resend integration
- Nodemailer integration
- Comprehensive documentation site
- MIT License

### Core Features
- `@weaver/email-core` - Core rendering logic
- `@weaver/email-components` - React email components
- `@weaver/email-vue` - Vue.js email components
- `@weaver/email-integrations` - Email provider adapters
